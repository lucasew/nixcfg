# Riverwood

Riverwood is my main laptop (Acer A315-51-51SL).

It features:
- Intel CPU/GPU
- 12GB RAM
- 1TB SSD
- Ext4 filesystem
- Dual booted with Windows 10
- Sway/i3 graphical environment

It is the first city you know after starting the game after the Helgen incident.

It is configured as a primary daily-driver workstation in the Nix/NixOS setup.
