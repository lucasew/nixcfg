# Riverwood (Laptop)

"First city you know after starting the game after the Helgen incident."

This directory contains the NixOS system configuration for the main daily-driver laptop (`Acer A315-51-51SL`).

## Specifications & Setup

- **Hardware:** Intel CPU/GPU, 12GB RAM, 1TB SSD
- **Filesystem:** ext4
- **Environment:** Sway / i3 window manager.
- **Boot:** Dual-boots with Windows 10.
