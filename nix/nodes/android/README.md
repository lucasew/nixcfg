# Android Node (Redmi Note 5 - `whyred`)

This directory contains scripts and notes specific to an Android device, specifically a Xiaomi Redmi Note 5 (codename `whyred`).

**Important:** This device is **not** managed by NixOS. The contents here are primarily useful shell scripts, configurations, and notes for applying manual tweaks or running utilities on the phone directly (e.g., via `adb` or Termux).
